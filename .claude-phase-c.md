### 3. `app/pomodoro.tsx` — Full Pomodoro Screen

Entry: modal/full screen pushed from anywhere.
Accept optional params: `taskId`, `label` (pre-filled if launched from a task).

**Layout:**
- Header: "Focus Session" + optional task context card (accent bg, task name)
- **Timer circle** — large SVG circular countdown using react-native-svg:
  - Circle fills clockwise as time passes (stroke-dashoffset animation using Reanimated)
  - Big center text: MM:SS remaining
  - Below center: "Focus" or "Break" label
- **Task label field** — BottomSheetTextInput-style (simple TextInput) — "What are you working on?" — required before starting, pre-filled if from task
- **Controls row:**
  - [Start / Pause] — large accent button (full width)
  - [Stop] — smaller secondary (only when running)
  - [+5 min] — ghost button (only when < 5 min remain)
- **Settings row** (below controls):
  - Focus: [15] [25] [45] [60] chips (default 25, disabled when running)
  - Break: [5] [10] [15] chips (default 5, disabled when running)
- **Today's stats bar** at bottom: "🍅 X sessions · Y min focused"

**Timer logic (in-component, using `useEffect` + `setInterval`)**:
- Start: record `startedAt = Date.now()`, call `pomodoro.startSession`
- Each second: compute `elapsed = (Date.now() - startedAt) / 1000`, `remaining = planned * 60 - elapsed`
- If remaining <= 0: complete phase (haptic burst, show break prompt)
- Pause: save `pausedAt`, clear interval
- Resume: adjust `startedAt += Date.now() - pausedAt`, restart interval
- Stop: call `pomodoro.interruptSession`, pop screen
- +5 min: add 300 to planned seconds
- Use `expo-haptics` ImpactFeedbackStyle.Heavy on completion

**On focus complete:**
- Show overlay/modal: "Session complete ✓" + "3 sessions today · 75 min"
- If task linked: "Mark [task] as done?" Yes/No buttons
- [Start break] → switches to break timer (5/10/15 min)
- [Done for now] → pops screen
